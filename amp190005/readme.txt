How to compile and run:

1. Unzip the folder and generate the java files
2. The main function is inside of the TestLP1.java
3. In the terminal, be on same level as parent folder of "amp190005", 
type following code to compile files Num.java, TestLP1.java 
	- javac ./amp190005/Num.java
	- javac ./amp190005/TestLP1.java
4. Type to execute the code

	- java -Xss512m -Xms2g amp190005.TestLP1 0
	- java -Xss512m -Xms2g amp190005.TestLP1 1
	- java -Xss512m -Xms2g amp190005.TestLP1 2
	- java -Xss512m -Xms2g amp190005.TestLP1 3
